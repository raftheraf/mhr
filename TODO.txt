COME ELIMINARE L’ERRORE 200 SU PHPMYADMIN: LA SOLUZIONE MIGLIORE
“C:\xampp“) e successivamente aprire la directory phpMyAdmin. Esiste un file “config.inc.php” contenente le impostazioni di PhpMyAdmin. Aprendolo con un editor testuale (evita il blocco note di Windows per evitare di diventare matto, meglio Sublime Text o Scite) e scorrendo le varie istruzioni (mantenendosi comunque all’inizio del file), si può trovare la seguente dicitura PHP (o una equivalente con un valore assegnato).

$cfg['Lang'] = 'it';
Fate diventare la precedente assegnazione come scritto qua sotto.

$cfg['Lang'] = 'en';
